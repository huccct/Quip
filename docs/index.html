Why Quip
Draft different angles, keep only the natural one

Quip first understands the post's language, intent, and tone. A writer × editor review flow removes misreadings, repetition, filler, and forced jokes before returning one editable draft.

Four reply styles

Adaptive, Funny, Warm, and Sharp—different expressions for the same post.

Matches the post's language

Chinese replies to Chinese, English replies to English, detected automatically.

Understands images and quotes

Uses images, video covers, and nested quoted posts, and clearly shows whether images were sent.

Choose your model

DeepSeek, OpenAI, Grok, or Claude with your own API key and credits.

You decide what gets posted

Quip only fills the reply draft. You review, edit, and send it yourself.

How it works
Install in three steps
Download and load

Download the zip and extract it. Open chrome://extensions, enable Developer mode, choose Load unpacked, and select the folder.

Choose a model and style

Open Quip, choose a provider and reply style, enter the matching API key, and save.

Open a post and reply

Click the ✦ button in the reply toolbar, review the generated draft, and send it yourself.
