怎么用
三步装好，一下就会
下载并加载

从 GitHub 下载解压，chrome://extensions 开「开发者模式」→ 加载已解压的扩展程序。

选模型、填 key

点 Quip 图标，选一个模型，填入它的 API key，保存。

点开推文，一键回

回复框工具栏会出现 ✦ 按钮，点它生成，改一改，手点发。

💡 目前需开发者模式加载。上架 Chrome 应用商店后，普通用户点一下就能装 —— 正在路上。
