为什么不一样
先从不同角度起草，再只留最自然的一句

Quip 让模型先理解原推的语言、意图和情绪，再用写手 × 编辑双视角自审淘汰误读、复述、废话和尬梗，最后只输出一条可直接修改的草稿。

四种回复风格

自适应、幽默、真诚、犀利；同一条推文，也能选择不同表达。

跟随原推语言

中文推回中文，英文推回英文，自动判断。

理解图片与引用推文

结合配图、视频封面和嵌套引用理解上下文，并明确提示图片是否随请求发送。

模型任选

DeepSeek、OpenAI、Grok、Claude，填自己的 key，回复走你的额度。

发不发你说了算

只把草稿填进回复框，由你修改和发送。推文内容及开启读取的图片会发给所选模型商。

怎么用
三步装好，一下就会
下载并加载

点上方「下载插件」得到 zip，解压。打开 chrome://extensions，开「开发者模式」→「加载已解压的扩展程序」→ 选解压出的文件夹。

选模型、风格，填 key

点 Quip 图标，选择模型和回复风格，填入对应 API key，保存。

点开推文，一键回

回复框工具栏会出现 ✦ 按钮，点它生成，改一改，手点发。
