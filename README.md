# Quip · 一键妙回复

在任意推文下，一键生成一条**有梗、像真人**的回复，填进回复框，你看一眼、手点发。

不是又一个满是翻译腔的 AI 回复工具 —— Quip 内置「写手 × 刷推老哥」双角色博弈：写手先写，老哥用「这条我会不会点赞」把无聊和尬玩梗的版本毙掉，只留下你真会发出去的那句。

## 特点

- **有钩子，不废话**：意外角度 / 反转 / 精准夸张 / 可共鸣的具体 / 反常识真话，挑最贴原推调性的
- **跟随原推语言**：中文推回中文，英文推回英文
- **多模型**：DeepSeek / OpenAI / Grok / Claude 任选，填自己的 key，回复走你自己的额度
- **真人点发**：只生成草稿填进回复框，发不发你说了算，不碰自动化 API，合规
- **纯前端**：零后端，key 只存在你自己浏览器里

## 安装（开发者模式）

1. 下载本仓库（Code → Download ZIP，解压）
2. 打开 `chrome://extensions`，右上角开启「开发者模式」
3. 点「加载已解压的扩展程序」，选这个文件夹
4. 点工具栏的 Quip 图标 → 选模型 → 填对应的 API key → 保存

## 用法

1. 在 x.com 点开某条推的**详情页**
2. 点回复框，工具栏图标那排右边会出现 Quip 的 ✦ 按钮
3. 点它 → 几秒后回复自动填进框 → 你改一改、手点发

## 说明

- key 只保存在浏览器本地（`chrome.storage.local`），不上传任何服务器
- 生成一条不满意？再点一次，自动换新的
- X 偶尔改版可能让按钮失效，欢迎提 issue
