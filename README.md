# Quip

**在任意推文下，一键生成一条有梗、像真人的回复。**

跟随原推语言 · 多模型 · 纯前端 · 真人点发

</div>

---

## 为什么不一样

市面上的 AI 回复工具，生成的中文一股翻译腔，一眼假。

Quip 内置**「写手 × 刷推老哥」双角色博弈**：写手先写，老哥用「这条我会不会点赞」把无聊的、尬玩梗的版本毙掉——只留下你真会发出去的那一句。

## 特点

- **有钩子** — 意外角度 / 反转 / 精准夸张 / 可共鸣的具体，挑最贴原推调性的
- **跟随语言** — 中文推回中文，英文推回英文
- **多模型** — DeepSeek · OpenAI · Grok · Claude，填自己的 key
- **你说了算** — 只填草稿进回复框，发不发你点，合规不碰自动化
- **纯前端** — 零后端，key 只存在你浏览器里

## 安装

```
1. 下载本仓库并解压
2. 打开 chrome://extensions，开启「开发者模式」
3. 「加载已解压的扩展程序」→ 选此文件夹
4. 点 Quip 图标 → 选模型 → 填 key → 保存
```

## 用法

在 x.com 点开某条推 → 点回复框 → 点工具栏的 <b>✦</b> → 回复自动填进框 → 改一改、发。

不满意？再点一次，自动换新的。
