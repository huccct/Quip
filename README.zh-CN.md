# Quip

**在任意推文下，一键生成一条贴合语境、像真人的回复。**

理解正文、配图与引用推文 · 四种风格 · 多模型 · 真人点发

## 为什么不一样

只让 AI「礼貌回复」，很容易得到正确但没必要发的废话。

Quip 会先判断作者是在分享、吐槽、自嘲、求助还是抛梗，再找到一个具体细节或潜台词来接话。内置的**写手 × 编辑双视角自审**会淘汰万能附和、误读、废话和尬梗，只输出最自然的一句。

## 特点

- **真正理解意思** — 围绕具体细节、潜台词、反差或上下文回扣接话，不复述原推
- **贴合语境** — 该接梗时有意思，严肃或难过时不硬抖机灵
- **风格可选** — 自适应、幽默、真诚、犀利
- **快捷重写** — 不用再开设置，一键改成更短、更自然或更有梗
- **贴近你的语气** — 填一段表达偏好或几条示例回复，指导措辞和视角
- **理解完整上下文** — 读取最近父级对话、配图、视频封面和嵌套引用推文
- **上下文预览** — 生成前查看 Quip 实际读取了什么
- **跟随语言** — 中文推回中文，英文推回英文
- **多模型** — DeepSeek、OpenAI、Grok、Claude、Gemini、OpenRouter，填自己的 Key
- **你说了算** — 只填草稿进回复框，由你修改和发送
- **零自建后端** — Key 保存在浏览器本地；推文内容及你开启读取的图片会直接发送给所选模型商

## 安装

```text
1. 从最新 GitHub Release 下载并解压 quip.zip
2. 打开 chrome://extensions，开启「开发者模式」
3. 「加载已解压的扩展程序」→ 选此文件夹
4. 点 Quip 图标 → 选模型 → 填 Key，设置会自动保存
```

[下载最新版本](https://github.com/huccct/Quip/releases/latest)

## 用法

在 x.com 点开某条推文 → 点回复框 → 点工具栏的 **✦** → 回复自动填进框 → 修改并发送。

不满意可直接点“更短 / 更自然 / 更有梗”，也可以再点一次 ✦ 重新生成。
